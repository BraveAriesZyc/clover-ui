# Icon 图标

## 安装

```ts
import { ZIcon, Icon, SvgIcon } from 'clover-ui'
```

## 用法

```vue
<ZIcon name="bx:home" />
<ZIcon icon="ri:leaf-line" size="md" />
<SvgIcon name="your-local-svg" />
```

## 属性概览（ZIcon）

- `name|icon` 图标 ID（Iconify）
- `size` 尺寸：`sm|md|lg|number(px)`
- `color` 颜色：可用主题色变量或自定义颜色值
