# Clover UI 文档

- 组件文档位于 `docs/components/`
- 按类别浏览：基础、表单、布局、图标、反馈

## 快速开始

- 安装样式：
  - 全量：`import 'clover-ui/styles.css'`
  - 按需：`import 'clover-ui/theme-chalk/variables.css'`
- 全量注册：`app.use(CloverUI)` 或按需注册单个组件

## 目录

- 基础：Button、Card
- 表单：Input
- 图标：Icon、SvgIcon
- 布局：ZLayout、ZFixedGrid、ZVirtualGrid、ZScrollChips
- 反馈：Loading、Message
